This plugin helps remove the Website URL field parameter from comment box in every WordPress themes.

== Description ==
"URL Field Remover From Comment Box" is a free plugin that helps you remove the Website URL field from comment box of every WordPress themes. By installing this plugin you can remove the website field from the comment form of every WordPress themes easily.

Features:

1. Remove URL Field from all free and premium WordPress themes.

== Installation ==
#### In the WordPress dashboard
1. Visit 'Plugins > Add New'
1. Search 'URL Field Remover From Comment Box'
1. Check the plugin is from 'Bimal Raj Paudel'
1. Install and Activate the plugin.
1. After activation you don't have to do anything. 

#### Manually
1. Download the file from <a href="https://wordpress.org/plugins/url-field-remover-from-comment-box/" target="_blank">HERE.</a>
1. Unzip the file.
1. Upload the folder URL Field Remover From Comment Box into 'wp-content/plugins/' folder.
1. Activate.

== Frequently Asked Questions ==
= How does the plugin work? =
The plugin injects the code directly to functions.php of your WordPress theme file to remove URL Field from the comment form.

= Will this plugin work for every WordPress themes? =
This plugin has been tested with many WordPress themes and seem to work with all of them. We will continue to test this plugin with many themes and update it, if it doesnt work with some of them.

= Will this break my theme? =
It should not break your theme unless it collides with any other plugins.

= Do you need to do anything after installing the plugin? =
No, you don't have to do anything. Just install and activate the plugin and it will do everything for you.

= Can I temporarily turn off the feature of this plugin? =
Yes. You can turn off the feature of this plugin by going to the settings of this plugin.
